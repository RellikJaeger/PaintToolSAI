时隔一年，小松浩司终于再次放出 SAI2 的进度报告版。
因为过两天就是我的大喜日子，就不完整翻译更新内容了，下面就着重点一下。

首先，终于加入了保存和打开功能，SAI2 终于可以实用了，但目前只有 .sai2 和 .bmp 格式，其中 .sai2 格式和 SAI1 的文件不兼容，但以后肯定是会支持的。

不过，打开和保存功能需要购买了正版才能使用，用破解的可能要等等了。
买了正版的需要重新下载授权证书，因为 SAI2 的系统 ID和 SAI1 不一样。

其他的新功能是钢笔图层和形状图层，矢量功能大升级。

最后，这次是跨年更新了两版，后一版看似是对上一版 BUG 的紧急修正，但我发现还是有些更新是作者没提到的，比如元旦版就多了英文语言，启用的方法是将 sai.ini 里的 ja 改成 en。
但如此一来，未来 SAI2 可能不会像SAI1 那样制作外挂语言文件，对汉化来说是个减分项。

如果你尝鲜运行过日文版的 SAI2，再运行汉化版，也会显示日文的笔刷，这时只需把下面的文件删除即可解决。
文档\SYSTEMAX Software Development\SAI2 Demo\settings.ssd

虽然功能还不是很完善，暂且算是比较能实用了，就先把软件发布出来吧。
由于是开发版本，自然可能会有BUG，如果希望使用可靠的软件的话，还是
请不要用这个开发版。此外，这个版本不是面向初级电脑用户的。如果

和 Windows 7 64bit，和上一版相比没有做太大的更改，所以 Windows 2000
以后的版本应该都能使用。

与2013-12-10版相比，增加的功能为以下内容。

打开/保存画布
在设置了许可证的情况下，打开/保存画布的功能变为可用。画布的
保存格式为 .sai2。这一格式不兼容 Ver.1 的格式。当前可用的保存
格式为 .sai2 和 .bmp 格式，我已经计划尽早支持其他格式。

2015-01-01 追记

这一版本中，当保存文件时会创建文件备份，并且将保留以往的备份文件。
请手动删除旧的备份文件。

钢笔图层
·开发了新的生成曲线的方法。与Ver.1相比能生成更平滑的曲线。
·从控制柄直接矢量化，可使用更少的控制点进行转换。
·实装了用修正液工具删除到交点的功能。
·实装了在贝塞尔曲线中使用控制柄的模式。(贝塞尔曲线模式意味着不使用新的曲线生成方法)
等，详细情况请实际使用看看。

形状图层
只是实装了基本的功能，和钢笔图层一样的操作就能创建形状。今后预定实装
的有控制柄的精密操作以及有限的SVG格式的输入和输出功能。

多视图
实装了与Ver.1相同的多视图功能。实装实验性质的从主窗口创建独立视图的功能。

自定义工具托盘
实装了与Ver.1相同的自定义/管理绘制工具的功能。

水彩边缘
实装了比以往的边缘色更鲜艳的“极彩边缘”模式。

BUG修复
·修复了笔划的出峰和入峰呈现阶梯状的问题。
·修复了笔工具的混色不正确的问题。
·修复的其他我记不住的各种问题。

关于许可证的设置
如果持有Ver.1的用户许可证，将许可证放入以下任何一个文件夹中，
即可使用打开/保存画布功能。但是，由于SAI2中系统ID的生成方法
已经改变，SAI2无法识别使用在Ver.1中显示的系统ID下载的许可证。
请使用在SAI2中显示的系统ID下载新的许可证放入以下的文件夹中。

·我的文档 → SYSTEMAX Software Development → SAI2 Demo 文件夹
 (包含文件 settings.ssd 的文件夹)

·SAI2的程序文件 sai2.exe 所在文件夹
