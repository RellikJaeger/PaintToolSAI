时隔一年，小松浩司终于再次放出 SAI2 的进度报告版。
因为过两天就是我的大喜日子，就不完整翻译更新内容了，下面就着重点一下。

首先，终于加入了保存和打开功能，SAI2 终于可以实用了，但目前只有 .sai2 和 .bmp 格式，其中 .sai2 格式和 SAI1 的文件不兼容，但以后肯定是会支持的。

不过，打开和保存功能需要购买了正版才能使用，用破解的可能要等等了。
买了正版的需要重新下载授权证书，因为 SAI2 的系统 ID和 SAI1 不一样。

其他的新功能是钢笔图层和形状图层，矢量功能大升级。

最后，这次是跨年更新了两版，后一版看似是对上一版 BUG 的紧急修正，但我发现还是有些更新是作者没提到的，比如元旦版就多了英文语言，启用的方法是将 sai.ini 里的 ja 改成 en。
但如此一来，未来 SAI2 可能不会像SAI1 那样制作外挂语言文件，对汉化来说是个减分项。

如果你尝鲜运行过日文版的 SAI2，再运行汉化版，也会显示日文的笔刷，这时只需把下面的文件删除即可解决。
文档\SYSTEMAX Software Development\SAI2 Demo\settings.ssd

|关于 SAI2
如今，SAI Ver.2 的开发正进展顺利。由于在 Ver.1 上新增功能的代价已经到头了，所以 Ver.1 的预定新增功能全部取消，它们将全部改为在 Ver.2 上实现。由于取消 Ver.1 的预定新增功能违背了以往的发言，作为补偿，Ver.1 的用户可无偿升级 Ver.2（无论 32bit/64bit）。
按照以上方针，原则上 Ver.1 以后只会有问题修正。敬请谅解。

|SAI2 的进度报告
这次发布的软件虽然是功能还不是很完善的半成品，但总算是比以前实用了。
由于是开发版本，自然很大可能会有BUG，如果想使用稳定的软件的话，还是请不要用这个开发版。此外，这个版本不是面向初级电脑用户的。如果不熟悉电脑的基本知识和操作的话，请不要使用这个版本。

由于测试的时间不多，这次只确认了能正常运行的系统为 Windows XP 32bit 和 Windows 7 64bit，相比于上一版没有太大的改变，所以 Windows 2000 及更新的版本应该都能用。
 
2015-01-01 追记
2014-12-31 版中有以下问题，所以发布了修正版。

·在混色的笔压计算中使用了未初始化的值。
（使用了椭圆尺的话会出现颜色淆乱，在正在绘画时也有可能发生）

·尝试多开软件会发生访问冲突。

·不小心忘了放刷毛的图像文件（圆笔和平笔）。
（后果是，如果您的解压缩软件不会解压缩空文件夹，则软件运行所需的文件夹将不会被创建，很可能导致启动时出错。）

2015-01-02 追记
2015-01-01 版中有以下问题，所以发布了修正版。

·透视尺的方向判断不正确。

·当合并的图层包含超出画布范围的部分时，发生访问冲突。

·按下部分按键时假死。

还有，由于可执行文件名与 Ver.1 同为 sai.exe，Wacom 的数位版驱动可能会工作不正常。现在已经把可执行文件名改为 sai2.exe，也许以往部分用户报告的数位板不正常工作的问题会得到解决。

与2013-12-10版相比，增加的功能为以下内容。

打开/保存画布
在设置了授权证书的情况下，打开/保存画布的功能变为可用。画布的保存格式为新的格式 .sai2。该格式不兼容 Ver.1。目前只可用 .sai2 和 .bmp 格式，我会计划尽早支持其他的保存格式。

2015-01-01 追记

在这一版本，保存文件时会创建文件备份，且不会自动删除以往的备份。
麻烦您手动删除旧的备份。

钢笔图层
·开发了新的曲线生成方法。能创建出比 Ver.1 更平滑的曲线。
·可使用更少的锚点将手绘线转换成矢量化的线段。
·在修正液工具中实装了删除到交点的功能。
·在贝塞尔曲线的图柄中实装了控制模式。（注意：贝塞尔曲线模式意味着不使用新的曲线生成方法）
等等，详细情况请实际使用看看。

形状图层
虽然目前只是实装了基本的功能，但拥有使用和钢笔图层一样的操作就能创建出形状的功能。
计划在以后实装锚点的精密操作，以及有限的 SVG 格式的输入和输出等功能。

多视图
现在可以操作与  Ver.1 相同的多视图功能。实装了试验性质的功能，能创建从主窗口独立出来的视图。

自定义工具盒
现在可以跟 Ver.1 一样自由的创建和管理绘画工具。

水彩边界
实装了比以往的边界颜色更鲜艳的“极彩边界”模式。

BUG 修复
·修复了笔画的出锋和入锋呈现梯形的问题。
·修复了画笔工具的混色不正确的问题。
·修复的其他我记不清的各种问题。

关于授权证书的设置
如果您拥有 Ver.1 的用户许可证，将授权证书放入以下任何一个文件夹中，即可其永久打开/保存画布的功能。但是，由于 SAI2 中系统ID的生成方法已经改变了，SAI2 无法识别用 Ver.1 显示的系统 ID 来下载的授权证书。
麻烦您使用在 SAI2 中显示的系统 ID 下载新的授权证书，并放入到以下的文件夹中。

·我的文档 → SYSTEMAX Software Development → SAI2 Demo 文件夹
 (包含文件 settings.ssd 的文件夹)

·SAI2 的程序文件 sai2.exe 所在文件夹

关于将来
这一年我一心工作在软件的那些眼睛看不到的部分。得益于此，软件的底层部分的开发前进了一大步，如果没有特别的情况，将来会每隔 1～3 个月左右更新一次。
