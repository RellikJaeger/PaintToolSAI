SAI2 2015-01-02 进度报告版
============================================================
【软件名称】PaintTool SAI Ver.2
【版　　本】2015-01-02 进度报告版
【版权所有】SYSTEMAX Software Development
【官　　网】http://www.systemax.jp/ja/sai/devdept.html
【汉　　化】Xelloss
【汉化主页】http://saier.me/sai2

时隔一年，小松浩司终于再次放出 SAI2 的进度报告版。
因为过两天就是我的大喜日子，就不完整翻译更新内容了，下面就着重点一下。

首先，终于加入了保存和打开功能，SAI2 终于可以实用了，但目前只有 .sai2 和 .bmp 格式，其中 .sai2 格式和 SAI1 的文件不兼容，但以后肯定是会支持的。

不过，打开和保存功能需要购买了正版才能使用，用破解的可能要等等了。
买了正版的需要重新下载授权证书，因为 SAI2 的系统 ID和 SAI1 不一样。

其他的新功能是钢笔图层和形状图层，矢量功能大升级。

最后，这次是跨年更新了两版，后一版看似是对上一版 BUG 的紧急修正，但我发现还是有些更新是作者没提到的，比如元旦版就多了英文语言，启用的方法是将 sai.ini 里的 ja 改成 en。
但如此一来，未来 SAI2 可能不会像SAI1 那样制作外挂语言文件，对汉化来说是个减分项。

如果你尝鲜运行过日文版的 SAI2，再运行汉化版，也会显示日文的笔刷，这时只需把下面的文件删除即可解决。
文档\SYSTEMAX Software Development\SAI2 Demo\settings.ssd
